Function to adjust accounting data:

Parameters:
df_contabil (DataFrame): Input DataFrame containing accounting data.

Returns:
DataFrame: Adjusted accounting data.

What does the function do?

The function adjust_accounting takes a DataFrame df_contabil as input.
It makes adjustments to Cost and Expense accounts by copying the DataFrame, selecting relevant columns, renaming them, and filtering out rows where either the current year's value or the previous year's balance is non-zero.
Similarly, adjustments are made to Result accounts using the same process.
The results from both adjustments are concatenated into a final DataFrame, and column names are standardized.
The function returns the final DataFrame with adjusted accounting data.

O que a função faz?

A função adjust_accounting recebe um DataFrame df_contabil como entrada.
Ela faz ajustes nas contas de Custo e Despesa copiando o DataFrame, selecionando colunas relevantes, renomeando-as e filtrando linhas onde o valor do ano atual ou o saldo do ano anterior é diferente de zero.
Da mesma forma, são feitos ajustes nas contas de Resultado usando o mesmo processo.
Os resultados de ambos os ajustes são concatenados em um DataFrame final, e os nomes das colunas são padronizados.
A função retorna o DataFrame final com os dados contábeis ajustados.

def ajusta_contabil(df_contabil):
    
    # Adjusting Cost and Expense accounts
    df_contabil_contas = df_contabil.copy()
    df_contabil_contas = df_contabil_contas[['datafin', 'nomegrupo', 'nomeconta', 'valor', 'saldo_anterior']].rename\
    (columns= {'datafin':'competencia', 'nomeconta':'nome_conta', 'nomegrupo':'nome_grupo', 'valor':'valor_conta', 
               'saldo_anterior':'ano_anterior'}) 
    df_contabil_contas = df_contabil_contas[(df_contabil_contas['valor_conta']!= 0) | (df_contabil_contas['ano_anterior'] != 0)]
    
    # Adjusting Result accounts
    df_contabil_grupos = df_contabil.copy()
    df_contabil_grupos = df_contabil_grupos[['datafin','nomegrupo', 'valorgrupo', 'valorgrupoant']].rename\
    (columns= {'datafin':'competencia','nomegrupo':'nome_conta','valorgrupo':'valor_conta', 'valorgrupoant':'ano_anterior'})
    df_contabil_grupos['nome_grupo'] = df_contabil_grupos['nome_conta']
    df_contabil_grupos = df_contabil_grupos
    df_contabil_grupos = df_contabil_grupos[(df_contabil_grupos['valor_conta'] != 0) | (df_contabil_grupos['ano_anterior'] != 0)]
    
    # Concatenation and final formatting
    df_contabil_final = pd.concat([df_contabil_contas,df_contabil_grupos],ignore_index=True)
    df_contabil_final.rename(columns={'competencia': 'Competencia','nome_grupo':'Grupo de Contas',
                                      'nome_conta':'Conta','valor_conta':'Ano Atual',
                                      'ano_anterior':'Ano Anterior'}, inplace=True)
    
    return df_contabil_final
